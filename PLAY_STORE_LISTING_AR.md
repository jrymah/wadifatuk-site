# نصوص متجر التطبيقات — العربية

## العنوان (30 حرفاً كحد أقصى)
```
وظيفتك — بوابة الوظائف
```

## الوصف القصير (80 حرفاً)
```
اكتشف آلاف الوظائف في المملكة العربية السعودية مع إشعارات فورية لكل فرصة جديدة
```

تطبيق خفيف الوزن يعمل بسلاسة على جميع الأجهزة، حتى مع الاتصال البطيء.
